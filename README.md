## 📝 About the Project

This project is a **Weather Dashboard** developed to provide real-time weather information with a high-end visual experience. Inspired by modern dashboard trends, it uses **Glassmorphism** effects and a dark aesthetic to deliver data clearly and elegantly.

The application was fully refactored to **English** and features a dynamic forecast system for the upcoming days, ensuring a global and professional utility.

## ✨ Main Features

- [x] **Real-time Search:** Get instant data for any city in the world.
- [x] **Glassmorphism UI:** Modern blurred-background cards.
- [x] **Next 2 Days Forecast:** Stay updated with upcoming weather trends.
- [x] **Advanced Metrics:** Humidity, wind speed, visibility, pressure, and cloud coverage.
- [x] **Fully Responsive:** Optimized for desktops, tablets, and smartphones.
- [x] **Internationalized:** Completely translated into English.
